# Getting Started

## Run
- Install deps: `bun install`
- Start: `bun run dev`

## First-time setup
- Ensure `NEXT_PUBLIC_CONVEX_URL` is set (see `.env.local`).
- Open the app (default `http://localhost:3000`).

## Join the world
- The app signs you in **anonymously** automatically.
- In **World**, enter a name and click **Join**.

- **Move**: `W`, `A`, `S`, `D`

- **World**: shared map, loot nodes, stations
- **Players**: players in the current biome
- **Inventory**: your inventory stacks
- **Crafting**: recipes + craft queue
