# Glossary

## defId
A stable string identifier for materials/components/items/recipes/stations (e.g. `fibrous_wood_strips`).

## biomeId
A string biome identifier (currently `forest`).

## stationId
A station identifier (e.g. `campfire`, `basic_workbench`).

## recipeId
A recipe identifier (e.g. `craft_crude_torch`).

## lootSourceId
A loot node type identifier (e.g. `fallen_branch`, `lore_cache`).

## craft job
An entry in the crafting queue with start/end times; outputs are granted when claimed.
