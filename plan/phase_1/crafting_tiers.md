# **FULL CRAFTING TECH TREE**

This tree follows the 5 crafting tiers and the material lifecycle (Raw → Refined → Formed → Component → Item). Each tier expands the player’s reach into new biomes, new techniques, and more intricate assemblies.

---

# **TIER 0 — HANDCRAFT BASICS**

Bare-hands crafting. Survival-level work.

### **Workstations**

* Ground crafting
* Makeshift fire pit
* Hand-shaping spot (no tool required)

### **Raw Materials Usable**

Fibers, basic sticks, stones, loose minerals, fungi, reeds.

### **Formed Materials**

* Fiber cord (hand-twisted)
* Reed strips
* Crude bark panels
* Sharpened stone flakes

### **Components**

* Primitive binding
* Hand-carved stake
* Simple sling strap

### **Items**

* Torch
* Basic satchel
* Crude knife
* Stone hammer
* Small shelter
* Rope ladder
* Tether line

This tier is the “scratching symbols into dirt” era.
